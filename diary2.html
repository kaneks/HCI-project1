Diary for Second Project

Project

This is record of the redesign light project. The purposes of the project are understand user need, emphatize user, brainstorm and then develop the prototype.

The Finished Project

This is the video of the finished project.

The Interview

So the first thing we do in this project is to understand more about user. We interview some users and the result are users like warm color when they want to relax. And they need light when they wake up in the middle of the night and want to go to toilet. Also, some user said that they don't like a bright light when they wake up in the middle of the night.

Brainstorm & Story Board

So after we get data from the user, we start to brainstorm the idea and come up with the solution. And how we will implement the solution

The Solution

The solution are having a light that can connect to the pressure sensor and when we put our feet on the floor the light will shining gradually.

Development

We using Philip Hue to create our prototype to control the temperature of the light and time Philip Hue use to lucent from the lowest level to highest level.

Feedback

After we finished the development then we go back and asking user and what he thinking about our prototype. He like it because he can use it in the real situation. He live in school dorm and when he turn off the light is completely dark and it hard for him to find the key card and go out to the toilet. If our prototype is apply it gonna be useful for him.

Reflection

What we learn from this project are we learn how to empathize the user, learn how to understand the user, know what user need. And then apply it to our design.
